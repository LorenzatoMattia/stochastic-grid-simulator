# Simulazione Incendio Cellulare

Una simulazione interattiva di un automa cellulare che modella la propagazione e l'estinzione di un "incendio" su una griglia.

## Demo Live

Puoi vedere l'applicazione funzionante qui: [https://tuo-username.github.io/nome-del-tuo-repository/](https://tuo-username.github.io/nome-del-tuo-repository/)

## Come Avviare Localmente

Segui questi passaggi per clonare il repository e avviare l'applicazione sul tuo computer:

1.  **Clona il repository:**
    ```bash
    git clone https://github.com/tuo-username/stochastic-grid-simulator.git
    cd stochastic-grid-simulator
    ```
2.  **Installa le dipendenze:**
    ```bash
    npm install
    # oppure con yarn
    # yarn install
    ```
3.  **Avvia l'applicazione in modalità sviluppo:**
    ```bash
    npm run dev
    # oppure con yarn
    # yarn dev
    ```
    L'applicazione sarà disponibile su `http://localhost:5173` (o una porta simile).
